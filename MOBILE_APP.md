# Мобильное приложение BIBLIOTECH

Это PWA-приложение. Его не нужно публиковать в App Store или Google Play: телефон устанавливает приложение прямо из сайта Render.

## Как оно подключено к общей базе

Приложение использует тот же backend и те же API-адреса:

- `/api/auth`
- `/api/books`
- `/api/comments`
- `/api/stats`

Если сайт запущен на Render с `DATABASE_URL`, и сайт, и установленное приложение работают с одной PostgreSQL-базой. Данные книг, пользователей, комментариев и статистики общие.

Service worker не кеширует `/api/*`, поэтому приложение всегда запрашивает свежие данные из общей базы.

## Установка на iPhone

1. Откройте сайт Render в Safari.
2. Нажмите кнопку `Поделиться`.
3. Выберите `На экран Домой`.
4. Нажмите `Добавить`.
5. На экране появится иконка BIBLIOTECH.

## Установка на Android

1. Откройте сайт Render в Chrome.
2. Нажмите меню с тремя точками.
3. Выберите `Установить приложение` или `Добавить на главный экран`.
4. Подтвердите установку.

## Важно

Для камеры, QR и установки приложения нужен HTTPS. Render-ссылка уже HTTPS, поэтому после деплоя приложение будет работать правильно.

Иконка сайта и логотип в интерфейсе меняются сразу при смене темы. Иконка уже установленного приложения на iPhone/Android может остаться старой из-за кеша системы; чтобы заменить ее, удалите ярлык BIBLIOTECH с главного экрана и добавьте приложение заново после выбора нужной темы.
