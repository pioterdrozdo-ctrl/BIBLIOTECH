# Парные режимы тем BIBLIOTECH

Система оформления хранит два независимых значения:

- `theme` — выбранная палитра (`light`, `dark`, `forest`, `ocean`, `sunset`, `violet`, `coffee`, `mono`);
- `bibliotech_theme_mode` — яркость выбранной палитры (`light` или `dark`).

Плавающая кнопка солнца/луны меняет только `bibliotech_theme_mode` и не заменяет выбранную палитру. Например, `ocean + dark` переключается в `ocean + light`.

Старые настройки мигрируют автоматически: ранее тёмные палитры открываются в тёмном режиме, ранее светлые — в светлом. После первого запуска режим сохраняется отдельно.
